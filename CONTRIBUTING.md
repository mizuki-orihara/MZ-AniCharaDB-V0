# Contributing to MZ-AniCharaDB

MZ-AniCharaDBにご興味を持っていただきありがとうございます。
このプロジェクトは現在、個人の学習と開発を目的としており、以下のポリシーで運用しています。

- **Pull Requests**: 基本的に受け付けていません。ソースコードの修正は作者本人が行います。
- **Issues**: バグの報告や、機能のアイデア、データの誤りなどは大歓迎です！お気軽に投稿してください。
- **Cloning**: 自分のローカル環境で試すためのクローンは自由に行っていただいて構いません。

作者のペースでじっくり育てていきたいため、ご理解いただけますと幸いです。
